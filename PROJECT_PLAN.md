Campus Connect Project Plan
Project Name
Campus Connect

Subtitle
Centralized event planning and registration platform for college clubs.

Main Idea
In college, different clubs conduct different events. Campus Connect provides one centralized system where:

Clubs create events
Super admin approves event schedules
Club admins publish approved events
Students register for published events
Admins manage attendance, payments, results, certificates, and reports
Final Workflow
text

Club Admin creates event
↓
Event becomes Pending Approval
↓
Super Admin reviews date, venue, time, and conflicts
↓
Super Admin approves/rejects/requests changes
↓
Club Admin publishes approved event
↓
Students view published event and register
User Roles
Super Admin
View all club events
View college-wide event calendar
Approve/reject events
Detect date and venue conflicts
Manage clubs and club admins
Club Admin
Create events for own club
View own club event calendar
Publish approved events
Manage registrations
Verify payments
Mark attendance
Coordinator
Verify participants
Verify payment
Mark attendance
Help manage event day operations
Student / Participant
View approved and published events
Register individually or as a team
Submit payment details
Submit feedback
