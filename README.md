Campus Connect
Campus Connect is a college club event management platform designed to manage events conducted by different clubs in a college.

Current Status
Step 1 completed: Frontend landing page UI.

Project Concept
Campus Connect allows:

Clubs to create events
Super admins to approve events
Club admins to publish approved events
Students to view and register for published events
Super admins to view all college events in a calendar
Club admins to view their own club events in a calendar
Planned Features
Public Student Side
View approved and published events
Search events
Filter by club
View event details
Register individually or as a team
Super Admin Side
Login
View all clubs
View all college events
College-wide event calendar
Approve/reject/request changes for events
Detect venue/date conflicts
Club Admin Side
Login
Create events
View own club calendar
Publish approved events
Manage registrations
Verify payments
Mark attendance
Coordinator Side
Verify payment
Verify ID/team details
Mark attendance

Workflow
Club admin creates an event.
Event status becomes pending approval.
Super admin approves or rejects the event.
If approved, club admin can publish the event.
Students can view and register only for published events.
